# ⚽ Club Atletico Carcaraña — Estadísticas

Aplicación web para gestionar las estadísticas del **Club Atlético Carcarañá**, desarrollada como proyecto personal.

## 🚀 Demo

Abrí `index.html` en cualquier navegador. No necesita servidor ni instalación.

## ✨ Funcionalidades

- **Plantel** — Agregar, editar y eliminar jugadores con nombre, posición y número
- **Partidos** — Cargar resultados con cálculo automático de victoria/empate/derrota
- **Goles y asistencias** — Registrar estadísticas individuales por partido
- **Tabla de goleadores** — Ranking automático con asistencias
- **Filtros** — Filtrar historial por competencia (Liga, Copa, Amistoso, etc.)
- **Estadísticas globales** — Partidos, victorias, empates, derrotas, goles y % de victorias
- **Persistencia** — Los datos se guardan en el navegador (localStorage)

## 🛠️ Tecnologías

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura de la página |
| CSS3 | Estilos, variables, diseño responsive |
| JavaScript (ES6+) | Lógica, manejo de datos, DOM |
| localStorage | Persistencia de datos en el navegador |

## 📖 Cómo usar

1. Cloná o descargá el repositorio
2. Abrí `index.html` en tu navegador
3. Empezá agregando jugadores al plantel
4. Cargá los partidos jugados
5. Registrá los goles y asistencias de cada partido
